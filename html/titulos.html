K19 - Desenvolvimento Web com HTML, CSS e JavaScript

Atualmente, particamente todos os sistemas corporativos possuem interfaces web. Para quem deseja atuar no mercado de desenvolvimento sde software, é obrigatório o conhecimento das linguagens: HTML, CSS e JavaScritp.

Essas linguagens são utilizadas para o desenvolvimento da camada de apresentação das aplicações web.

Pré-requisitos

  • Familiaridade com algum sistema operacional (Windows/Linux/Mac OS X)
  • Lógica de programação
